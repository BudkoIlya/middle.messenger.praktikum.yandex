## Установка

- `npm install` — установка зависимостей,
- `npm run dev` — запуск в режиме разработки,
- `npm run start` — запуск в режиме превью,
- `npm run prebuild` — запуск prettier, lint, stylelint, tsc.
- `npm run build` — сборка.
